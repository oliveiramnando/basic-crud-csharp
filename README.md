# Learning C#

This repository documents my progress learning **C# and .NET**, beginning with the fundamentals of the C# language and gradually moving into backend and API development.

The goal is to build a strong understanding of C# rather than only following tutorials. Each folder contains exercises or small projects focused on a particular concept.

## What I’m Learning

* C# syntax and language fundamentals
* Variables, data types, and operators
* Conditional statements and loops
* Methods and classes
* Object-oriented programming
* Collections and generics
* Exception handling
* Asynchronous programming
* REST API development with ASP.NET Core
* Database integration with SQL Server
* Entity Framework Core
* Testing and backend architecture

## API Projects

The `api-projects` directory contains small backend applications created to practice concepts such as:

* CRUD operations
* HTTP methods and status codes
* Request and response models
* Dependency injection
* Controllers and routing
* Service and repository layers
* Database persistence
* Validation and error handling

## Running a Project

Make sure the [.NET SDK](https://dotnet.microsoft.com/download) is installed.

Navigate into the project you want to run:

```bash
cd api-projects/TodoApi
```

Restore its dependencies:

```bash
dotnet restore
```

Run the application:

```bash
dotnet run
```
